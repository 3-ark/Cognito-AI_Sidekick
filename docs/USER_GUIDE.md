# Cognito: Your AI Memory Assistant - User Guide

## Introduction

Welcome to Cognito, your AI-powered memory assistant for your browser. Cognito is designed to help you think less and recall more by remembering everything you read, see, and do online. This guide will walk you through the features and functionalities of Cognito.

## Features

Cognito is packed with features to help you capture, organize, and recall information from the web.

### Connecting to AI Models

Cognito can connect to a variety of AI models, both local and cloud-based. You can configure these connections in the "Connect" tab of the settings.

*   **Local Models:**
    *   **Ollama:** Connect to a running Ollama instance on your local machine.
    *   **LM Studio:** Connect to a running LM Studio instance on your local machine.
*   **Cloud Services:**
    *   **Groq:** Connect to the Groq API using your API key.
    *   **Gemini:** Connect to the Google Gemini API using your API key.
    *   **OpenAI:** Connect to the OpenAI API using your API key.
    *   **OpenRouter:** Connect to the OpenRouter API using your API key.
*   **Custom Endpoints:** You can also configure your own custom endpoints to connect to other AI models.

### Note-Taking

Cognito's powerful note-taking system allows you to save and organize information from the web.

*   **Create, Edit, and Delete Notes:** You can create notes with a title, content (in Markdown), and tags. You can also edit and delete existing notes.
*   **Rich Text Editing:** The note editor supports Markdown, including tables, code blocks, math equations, and wiki-style links to other notes.
*   **Search:** You can search for notes by title, content, and tags.
*   **Import and Export:**
    *   Import notes from a variety of file formats, including `.txt`, `.md`, `.html`, `.pdf`, `.csv`, `.json`, `.zip`, and `.epub`.
    *   Export your notes to Markdown files, which are compatible with Obsidian.
*   **Browser Integration:**
    *   Create a new note from the content of the current page.
    *   Add selected text from any webpage directly to a new or existing note.

### Settings

*   **Text-to-Speech (TTS):**
    *   Have Cognito read your notes or other text aloud.
    *   Choose from a variety of system voices.
    *   Adjust the speech rate to your liking.
*   **Page Context:**
    *   Control the amount of text from a webpage that is sent to the AI model as context.
